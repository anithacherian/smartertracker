# SmartER Queue & Resource Tracker

A Django-based Emergency Room Queue and Resource Management System with role-based authentication for Admin, Receptionist, Doctor, and Nurse.

## Features

- Role-based Login
- Patient Management
- ER Queue Management
- Resource Management
- Doctor Dashboard
- Nurse Dashboard
- Receptionist Dashboard
- Admin Dashboard
- Doctor Status Updates
- Dashboard Statistics

## Tech Stack

- Python
- Django
- SQLite
- HTML
- CSS
- Bootstrap
- AdminLTE
- JavaScript
- jQuery

## Installation

1. Clone the repository
2. Create a virtual environment
3. Install dependencies
4. Run migrations
5. Start the development server
